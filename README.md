# 🎓 Projeto: Site Escolar com Bootstrap

## 📌 Sobre o Projeto

Este projeto consiste no desenvolvimento de um site institucional para uma escola fictícia, com o objetivo de praticar e aprimorar habilidades em **HTML, CSS e Bootstrap**.

A proposta é simular um ambiente real de desenvolvimento, aplicando conceitos de **responsividade, organização de layout e boas práticas de front-end**.

---

## 🚀 Objetivos

* Praticar o uso do **Bootstrap 5**
* Criar layouts **responsivos** (mobile e desktop)
* Trabalhar com **componentes prontos** do Bootstrap (navbar, botões, grids, etc.)
* Melhorar a organização de código HTML e CSS
* Simular um projeto real de sistema institucional

---

## 🛠️ Tecnologias Utilizadas

* HTML5
* CSS3
* Bootstrap 5
* Bootstrap Icons

---

## 📷 Layout Responsivo

O site foi desenvolvido com foco em **responsividade**, garantindo uma boa experiência em:

* 📱 Dispositivos móveis
* 💻 Computadores

---

## 💡 Observações

Este projeto tem caráter **educacional** e foi desenvolvido como forma de aprendizado e evolução na área de desenvolvimento web.
